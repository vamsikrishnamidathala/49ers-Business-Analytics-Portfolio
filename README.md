# San Francisco 49ers Business Analytics Work Sample

### Project Overview
This project analyzes 2024 NFL play-by-play data to identify high-leverage moments and offensive trends for the San Francisco 49ers. The goal is to demonstrate how raw data can be translated into actionable insights for Marketing, Content, and Business Operations teams.

### Business Goal
To move beyond standard box-score statistics and utilize advanced metrics like Expected Points Added (EPA) and Win Probability Added (WPA) to inform content strategy and maximize fan engagement.

### Methodology
* **Data Source:** Official NFL play-by-play data ingested directly via Python (Pandas) from the `nflverse` repository.
* **Tools:** Python, Pandas, Matplotlib, Seaborn.
* **Analysis:**
    * Calculated average EPA by play type to assess offensive efficiency trends.
    * Isolated top plays by WPA to identify highest-emotion marketing moments.
